# 🌸 AniStream — Anime Streaming & Discovery Platform

A modern anime discovery platform. Search anime, view detailed info, watch trailers, browse seasons, check voice cast, and get streaming links — all in one clean UI.

![Next.js](https://img.shields.io/badge/Next.js-black?style=flat&logo=next.js)
![TypeScript](https://img.shields.io/badge/TypeScript-blue?style=flat&logo=typescript)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-38bdf8?style=flat&logo=tailwindcss)

---

## ✨ Features

- 🔍 **Search Anime** — fast, fuzzy search across thousands of titles
- 📖 **Detail Pages** — synopsis, genre, status, rating, episode count
- 🎬 **Trailers** — embedded trailer playback per title
- 📅 **Seasons** — season-wise breakdown (current, upcoming, past)
- 🎙️ **Voice Cast Info** — voice actor names + character mapping
- ▶️ **Streaming Links** — direct links to watch episodes
- 📱 **Responsive UI** — clean, anime-inspired design, mobile-first

---

## 🛠️ Tech Stack

**Frontend:** Next.js · TypeScript · Tailwind CSS
**Data Layer:** Custom scraper service (Node.js)
